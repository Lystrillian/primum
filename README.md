# Primum

Primum means first from latin, do the math.

## Composition

This starter kit is composed of:
- Next.JS (App Router);
- Lenis
- Theatre.js
- react-three-fiber
- Tailwind configuration and plugins:
- - Typography
- - Container Queries
- - Animate
- - Prettier
